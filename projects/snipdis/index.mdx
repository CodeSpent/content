## Overview

SnipDis is a feature-rich Discord bot designed to make sharing and managing code snippets effortless for development teams. Whether you're collaborating on a project, helping others debug, or building a knowledge base, SnipDis provides the tools you need.

## Key Features

### Syntax Highlighting
Support for over 150 programming languages with beautiful syntax highlighting powered by Prism.js.

### Version Control
Track changes to your snippets over time with built-in versioning. Roll back to previous versions or compare differences.

### Team Collaboration
- Share snippets privately or publicly within your server
- Organize snippets with custom tags and categories
- Search through your team's snippet library

### Smart Search
Quickly find the snippet you need with full-text search across titles, descriptions, and code content.

## Getting Started

1. Click the "Add to Discord" button above
2. Select your server and authorize the bot
3. Use `/snippet create` to create your first snippet
4. Share it with `/snippet share <name>`

## Commands

```
/snippet create   - Create a new code snippet
/snippet list     - View all your snippets
/snippet share    - Share a snippet in the channel
/snippet edit     - Edit an existing snippet
/snippet delete   - Delete a snippet
/snippet search   - Search for snippets
```

## Technical Details

Built with modern technologies to ensure reliability and performance:

- **Node.js** for the runtime environment
- **Discord.js v14** for Discord API integration
- **MongoDB** for snippet storage
- **Redis** for caching and rate limiting
- **TypeScript** for type safety
